# NizarBootstrap
Isolating Bootstrap from other components in frameworks to avoid Bootstrap css conflict
```html
<html>
            ```
            ```html
  <!-- here there is no bootstrap style -->
  ```
  ```html
  <div class="NizarBootstrap>
```html
            <!-- here there is bootstrap style -->
            ```
```html
</div>
```
```html
<!-- here there is no bootstrap style -->
```
```html
</html>
```
